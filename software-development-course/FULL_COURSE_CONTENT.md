# Full Stack Development Bootcamp - 12 Week Curriculum

## Overview
A comprehensive 12-week bootcamp teaching Full Stack Development from HTML/CSS fundamentals to deploying production applications. Each week follows the structure: 3 lectures, 1 lab, 1 quiz, with 4 exams at strategic intervals.

---

## Weekly Breakdown

### Week 1: HTML & CSS Fundamentals
**Objective:** Master the building blocks of web pages

**Lectures:**
- Monday: HTML Structure - Semantic HTML, document structure, forms, accessibility
- Wednesday: CSS Basics - Selectors, box model, flexbox, responsive design
- Friday: CSS Advanced - Grid layout, animations, transitions, custom properties

**Lab:** Build Your First Website
- Create a responsive personal portfolio website with 3+ pages
- Apply semantic HTML and modern CSS techniques
- Ensure mobile responsiveness

**Quiz:** HTML & CSS Quiz (50 points)
- Test semantic HTML understanding
- CSS layout and responsive design knowledge

---

### Week 2: JavaScript Essentials
**Objective:** Learn JavaScript fundamentals and DOM manipulation

**Lectures:**
- Monday: JavaScript Basics - Variables, data types, operators, control flow
- Wednesday: Functions & Scope - Declarations, arrow functions, closures, scope
- Friday: DOM Manipulation - Select, create, modify, and delete DOM elements

**Lab:** Todo List App
- Build a dynamic todo application with JavaScript
- Implement add, delete, edit functionality
- Use local storage for persistence

**Quiz:** JavaScript Fundamentals Quiz (50 points)

---

### Week 3: JavaScript Advanced
**Objective:** Master asynchronous JavaScript and API handling

**Lectures:**
- Monday: Promises & Callbacks - Callback functions, promises, async patterns
- Wednesday: Async/Await - Master async/await syntax and error handling
- Friday: REST APIs & Fetch - Call external APIs using fetch API

**Lab:** Weather App API
- Build an app that fetches weather data from OpenWeatherMap API
- Display data dynamically with proper error handling
- Implement search functionality

**Quiz:** Async JavaScript Quiz (50 points)

---

### Week 4: React Basics
**Objective:** Learn React fundamentals and component-based architecture

**Lectures:**
- Monday: React Fundamentals - JSX, components, props, composition
- Wednesday: React State - useState hook, state management, re-rendering
- Friday: React Events & Forms - Handle input, form submission, events

**Lab:** React Component Library
- Build 8+ reusable React components
- Proper prop documentation and usage examples
- Deploy component library

**Quiz:** React Basics Quiz (50 points)

**EXAM 1 (Week 4):** Frontend Fundamentals
- 120 minutes, 200 points
- Covers HTML, CSS, JavaScript, React basics
- Mix of theory and coding challenges

---

### Week 5: React Advanced
**Objective:** Master advanced React patterns and performance

**Lectures:**

- Friday: Performance Optimization - Memoization, code splitting, lazy loading

**Lab:** React Ecommerce Store

**Quiz:** React Hooks Quiz (50 points)

**Lectures:**

**Objective:** Build and deploy a complete full-stack application

**EXAM 4 (Week 12):** Final Comprehensive Assessment
- 180 minutes, 250 points
- Covers all 12 weeks of material

## Grading System

### Final Score Calculation
- **Labs: 40%** - Practical implementation and coding skills
- **Quizzes: 30%** - Knowledge assessment (1 quiz per week = 600 total points)
- **Exams: 30%** - Comprehensive assessments (4 exams = 850 total points)

### Passing Requirements
- **Minimum Final Score: 70%** for certificate
- **Lab Submission:** Required for all weeks
- **Attendance:** Full participation in all lectures
- **Academic Integrity:** Zero tolerance for plagiarism

### Penalties
- **Plagiarism:** 3-strike system
  - Strike 1: Warning + score reduced to 50%
  - Strike 2: Further 10% reduction from final grade
  - Strike 3: Certificate ineligible

## Adding Content via Content Manager

### Access Content Manager
1. Log in as instructor
2. Click avatar (top right) → Dashboard
3. Navigate to "Content Manager"

### Add Lectures
1. Select Week (1-12)
2. Click "Lectures" tab
3. Fill in:
   - **Title:** e.g., "Monday: HTML Structure"
   - **Video URL:** YouTube/Vimeo embed link
   - **Description:** Topic overview
   - **Content:** Optional lecture notes/resources
4. Click "Add Content"

### Add Labs
1. Select Week (1-12)
2. Click "Labs" tab
3. Fill in:
   - **Title:** Lab name
   - **Description:** What students build
   - **Instructions:** Step-by-step requirements
4. Assignments automatically set to 100 points

### Add Quizzes
1. Select Week (1-12)
2. Click "Quizzes" tab
3. Fill in:
   - **Title:** Quiz name
   - **Description:** Topics covered
4. Add questions via quiz editor (50 points default)

### Add Exams
1. Only for Weeks 4, 8, 10, 12
2. Click "Exams" tab
3. Add exam questions with multiple choice or coding options
4. Questions automatically scored based on point allocation

## Content Guidelines

### Lectures
- Keep each lecture 45-60 minutes
- Include practical code examples
- Use interactive Q&A sessions
- Record for async viewing

### Labs
- 1 lab per week, due by Friday
- Clearly define requirements and acceptance criteria
- Provide starter code or templates when helpful
- Include plagiarism checks via code similarity detection

### Quizzes
- 8-10 questions per week
- Mix of multiple choice and short answer
- Focus on key concepts
- Auto-graded for immediate feedback

### Exams
- Comprehensive assessments
- Include coding challenges
- Time-limited to prevent cheating
- Manual review of complex responses

## Plagiarism Detection

### Automated Checks
- Code similarity analysis for all lab submissions
- Threshold: >70% similarity flags for review
- Database of student submissions compared
- External source detection via Turnitin API

### Strike System
- **Academic Integrity Violations:**
  - Copy-paste from external sources
  - Identical code to other students (>85% match)
  - Unauthorized collaboration
  
### Consequences
- Email notification to student
- Record in student profile
- 3 strikes = certificate ineligibility
- Potential course failure
